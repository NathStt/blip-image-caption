# 🖼️ BLIP Image Caption Generator

This project demonstrates how to generate image captions using the **BLIP (Bootstrapping Language-Image Pretraining)** model from Hugging Face Transformers.

It loads an image, processes it using a pretrained vision-language model, and generates a natural language description of the image.

---

## 🚀 Features

- Generate captions from any image
- Uses pretrained BLIP model from Hugging Face
- Simple and minimal Python implementation
- Works with local images

---

## 🧠 How it works

1. Load a pretrained BLIP model
2. Open and preprocess an image
3. Convert image into model inputs
4. Generate caption using the model
5. Decode output into human-readable text
